# 🀄 claude-spinners-zh

> 用中文互联网文化替换 Claude Code 加载词——让每次 AI 思考都充满梗

[![Claude Code](https://img.shields.io/badge/Claude%20Code-v2.1.23%2B-orange)](https://github.com/anthropics/claude-code)

---

Claude Code 跑任务时会显示加载词（spinnerVerbs），默认是英文。这个项目提供中文主题词包，复制粘贴即可使用。

## 使用方法

打开 `~/.claude/settings.json`，把想要的主题词块粘贴进去，重启 Claude Code 即生效。

> 需要 Claude Code **v2.1.23+**（2026-01-28 发布），低于此版本配置会被忽略。
